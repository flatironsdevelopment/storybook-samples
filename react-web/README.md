## üìù Minimal Requirements

- NodeJs 14.x
- Yarn

## Getting Started

Just install the dependencies with yarn

```bash
yarn
```

Then, start the storybook server.

```bash
yarn storybook
```

Storybook will automatically opens your browser :)
